Physical Computation | 2022

College Group Project

Co-creators: Anna Pragman, Krithi Nalla

Roles: project development, conceptualisation, garment testing, arduino coder, welding

For this project, we wanted to work with wearable-tech to build a tactile, experimental and performative installation piece. Using soft sensors and found materials, we created a garment that was an intervention model in a speculative future. In this future, following a bio-arm war, humans no longer touched one another. The garments were built (by an outsider) to encourage curious interactions and human connection amongst these humans
